# AI Resume Analyzer Using Gemini AI

## Features

- Resume Parsing
- ATS Analysis
- Job Description Matching
- NLP Similarity Score
- Skill Extraction
- Missing Skills Detection
- Interview Question Generation

## Tech Stack

- Python
- Streamlit
- Gemini AI
- Scikit-Learn
- PyPDF2

## Run Locally

```bash
pip install -r requirements.txt
streamlit run app.py
```
